JeeTrix
=======

This files describe the JeeTrix project.


GENERAL NOTES
=============

JeeTrix is a Tetris clone, in Java. It was started as an academic project, but since the
subject was really fun, it turned into a fully playable game.

It's GPLv3 licenced.


HOW TO PLAY
===========

Everybody knows how to play Tetris : your goal is to make line with new arriving blocks at 
the top.

Launch the game then choose your gameset (with 1, 2 or 3 keys) as your convinience.

Game keys
---------

Left / Right - move the current block
Up - rotate the current block
Down - move down 
Space - push down the current block
c - switch between the current block and the cached block

Escape - quit the game
F - switch windowed mode / fullscreen


REQUIREMENTS
============

A Java 7 JRE for run the game, as the corresponding JDK to compile it.
